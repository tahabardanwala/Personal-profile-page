# Personal Profile Page (HTML & CSS)

This repository contains a **prototype of a personal profile page** created using only **HTML and CSS**.
The project focuses on a simple, static front-end design to showcase personal information, layout structure, and styling without using JavaScript or any frameworks.

## Features

* Built with **HTML5** and **CSS3**
* Clean and minimal design
* Basic responsive layout
* Easy to customize
* No external libraries or frameworks

## Project Structure

```
├── index.html
├── style.css
└── README.md
```

* `index.html` – Main HTML structure of the profile page
* `style.css` – Styling for layout, colors, and typography

## Purpose

This project was created as:

* A **front-end practice prototype**
* A **personal profile / portfolio page layout**
* A demonstration of **HTML and CSS fundamentals**
